Go 1.8 Release Notes

  • Version 1.8 Release Notes
    • go1.8.1 (released 2017/04/07) includes fixes to the compiler, linker, runtime, documentation, go command and the crypto/tls, encoding/xml, image/png, net, net/http, reflect, text/template, and time packages. See the Go 1.8.1 milestone in the Go issue tracker for details.
    • go1.8.2 (released 2017/05/23) includes a security fix to the crypto/elliptic package. See the Go 1.8.2 milestone in the Go issue tracker for details.
    • go1.8.3 (released 2017/05/24) includes fixes to the compiler, runtime, documentation, and the database/sql package. See the Go 1.8.3 milestone in the Go issue tracker for details.
